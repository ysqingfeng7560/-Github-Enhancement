# Github Enhancement

## 🌐 Overview / 项目简介

**Github Enhancement** is a userscript that relies on public acceleration nodes to provide quick single-file downloads of Github project files and project lists.

**Github Enhancement** 是一款依赖于公益加速节点，提供 Github 项目文件及项目列表单文件快捷下载服务的用户脚本。

This script is designed for browser userscript managers such as **Tampermonkey** and **Violentmonkey**.

本脚本适用于 **Tampermonkey**、**Violentmonkey** 等浏览器用户脚本管理插件。
